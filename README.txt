####WEB SEC ANALYZER PROJECT###

Prerequisites:
•	User must be using Windows Operating System, XP or higher
•	User must have access to the internet to download my files
•	User must have Python pre-installed into their machine, with pip3 installed also
•	It is recommended to have XAMPP downloaded in order to host the index.html file, but HTML also works when access locally for viewing purposes. 



###HOW TO RUN PYTHON SCRIPTS###
1. enter command line, navigate to path where this folder is located
2. open python folder (cd python)
3. choose which python script to run (either py ipaddr.py or py xssscanner.py)
4. enter URL that you would like to scan
5. wait for results!
